# Prediction of Hospital Admissions from the Emergency Department

## Project Description
This web-based application predicts whether a patient from the emergency department requires hospital admission using data mining techniques.  
The project is implemented using JSP and Java to process patient data and generate predictions.

## Technologies Used
- Java
- JSP
- HTML, CSS, JavaScript
- Apache Tomcat
- MySQL
- Data Mining

## Key Features
- Patient data input through web forms
- Server-side processing using JSP
- Admission prediction based on data analysis
- Database connectivity for storing patient information

## How to Run
1. Import the project into Eclipse / NetBeans
2. Configure Apache Tomcat
3. Set up the database and update credentials
4. Run the project on the server
5. Access via browser
